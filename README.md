# Marketplace API - Seller Authentication

A secure RESTful authentication module for marketplace sellers, built with **NestJS**, **Prisma ORM**, and **MySQL**. The project implements seller registration, login, and logout using **JWT authentication**, **bcrypt password hashing**, and **request validation** following modern backend development best practices.

## Features

- Seller Registration
- Seller Login
- Seller Logout
- JWT Authentication
- Password Hashing with bcrypt
- Request Validation using class-validator
- MySQL Database with Prisma ORM
- Standardized HTTP Error Responses

## Tech Stack

- NestJS
- Prisma ORM
- MySQL
- JWT
- Passport
- Bcrypt
- TypeScript
